# FinTech---Fraud-Detection
Real-Time Anomaly Detection in Financial Transactions Project Overview Case Study: SecurePay FinTech Financial fraud is a multi-billion dollar problem. For a FinTech company, the ability to distinguish between a legitimate purchase and a fraudulent one in milliseconds is the difference between profit and reputational ruin.
However, fraud is statistically rare—often representing less than 0.1% of all transactions. This extreme class imbalance makes traditional supervised classification (like Logistic Regression) unreliable, as the model can simply guess "Legitimate" 100% of the time and still achieve 99.9% accuracy. As an AI Engineer, your initiative is to build an Unsupervised Anomaly Detection System. Instead of teaching the model what fraud looks like (which changes constantly), you will teach it what "normal" looks like.
